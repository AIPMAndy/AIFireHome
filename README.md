# 🔥 AIFireHome - AI 副业助手

> 用 AI 帮你找到适合自己的副业方向，7天启动计划一键生成

[![在线体验](https://img.shields.io/badge/🚀_在线体验-点击访问-blue)](https://aifirehome.netlify.app)
[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

## 🎯 这是什么？

很多人想做副业，但不知道：
- 自己适合做什么？
- 从哪里开始？
- 有什么风险？

AIFireHome 用 AI 帮你解决这三个问题。

## ✨ 核心功能

| 功能 | 说明 |
|------|------|
| 🔍 技能挖掘 | AI 分析你的背景，发现被忽视的变现技能 |
| 📊 副业匹配 | 基于你的时间、资源、风险偏好推荐副业方向 |
| 📅 7天启动计划 | 一键生成可执行的启动计划，不再拖延 |
| ⚠️ 风险评估 | 提前识别坑点，避免踩雷 |

## 🚀 立即体验

👉 **[点击访问在线版](https://aifirehome.netlify.app)**

或本地运行：
```bash
git clone https://github.com/AIPMAndy/AIFireHome.git
cd AIFireHome
npm install && npm run dev
```
